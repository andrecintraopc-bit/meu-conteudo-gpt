# meu-conteudo-gpt

Conteúdos e documentação prática, com foco em OpenClaw, agentes, workflows e handoffs.

## Estrutura
- OpenClaw/00_Inbox: coisas soltas para organizar depois
- OpenClaw/01_Setup: instalação, VPS, gateway, vars
- OpenClaw/02_Agentes: SOUL/BOOT/USER e políticas
- OpenClaw/03_Workflows: n8n, automações, integrações
- OpenClaw/04_Logbook: registro de sessões e decisões
- OpenClaw/99_Anexos: prints, exports, arquivos auxiliares
